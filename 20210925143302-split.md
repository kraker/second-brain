Allows **splitting** a string into multiple strings. By default splits at
_whitespace characters_.

```python
>>> 'My name is Simon'.split()
['My', 'name', 'is', 'Simon']
```

You can pass a different delimiter if you want.

```python
>>> 'MyABCnameABCisABCSimon'.split('ABC')
['My', 'name', 'is', 'Simon']
```

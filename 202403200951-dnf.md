# dnf

List repositories verbosely:

```bash
dnf repolist -v
```

List all available packages from all repos:

```bash
dnf repoquery
```

List all installed packages:

```bash
dnf list installed
```

List all installed packages and packages available for installation:

```bash
dnf list
```

List all available updates:

```bash
dnf list updates
```

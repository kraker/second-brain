---
title: islower()
date: 2021-09-25T14:23:00Z
tags:
- 'python'
---

Returns _True_ if string has at least one letter and all characters are
lowercase, otherwise returns _False_.

```python
>>> spam
'Hello, world!'
>>> spam.islower()
False
>>> 'abc12345'.islower()
True
>>> '12345'.islower()
False
```

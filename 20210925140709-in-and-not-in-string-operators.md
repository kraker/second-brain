Just like with [lists](20210913183709-list.md) you can use the **in** and **not
in** operators to check if something is in a string.

```python
>>> 'Hello' in 'Hello, World'
True
>>> 'HELLO' in 'Hello, World'
False
>>> '' in 'spam'
True
>>> 'cats' not in 'cats and dogs'
False
```

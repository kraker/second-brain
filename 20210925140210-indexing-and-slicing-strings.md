Strings use _indexes_ and _slices_ just like [lists](20210913183709-list.md).

```python
>>> spam = 'Hello, world!'
>>> spam[0]
'H'
>>> spam[-1]
'!'
>>> spam[0:5]
'Hello'
>>> spam[:5]
'Hello'
>>> spam[7:]
'world!'
```

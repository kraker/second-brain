# Tilde Expansion

| Prefix  | Value              |
| ------- | ------------------ |
| `~`     | `$HOME`            |
| `~user` | Home dir of _user_ |
| `~+`    | `$PWD`             |
| `~-`    | `$OLDPWD`          |
| `~N`    | `dirs +N`          |
| `~+N`   | `dirs +N`          |
| `~-N`   | `dirs -N`          |

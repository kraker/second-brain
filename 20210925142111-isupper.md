---
title: isupper()
date: 2021-09-25T14:21:00Z
tags:
- 'python'
---

Returns _True_ if string has at least one letter and all characters are
uppercase, otherwise returns false.

```python
>>> spam = 'Hello, world!'
>>> spam.isupper()
False
>>> 'HELLO'.isupper()
True
>>> '12345'.isupper()
False
```

Change a file's mode
Supports *octal* or *symbolic*


### File Modes

| Octal | Binary | File Mode |
| ----- | ------ | --------- |
| 0     | 000    | ---       |
| 1     | 001    | --x       |
| 2     | 010    | -w-       |
| 3     | 011    | -wx       |
| 4     | 100    | r--       |
| 5     | 101    | r-x       |
| 6     | 110    | rw-       | 
| 7     | 111    | rwx       |

### Symbolic Notation

| Symbol | Meaning      |
| ------ | ------------ |
| u      | user         |
| g      | group        |
| o      | others/world |
| a      | all = u+g+o  |

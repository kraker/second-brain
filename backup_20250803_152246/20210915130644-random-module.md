## Random lists

* `random.choice()`
* `random.shuffle()`

```python
>>> import random
>>> pets = ['Dog', 'Cat', 'Moose']
>>> random.choice(pets)
'Moose'
>>> random.choice(pets)
'Cat'
```

```python
>>> import random
>>> people = ['Alice', 'Bob', 'Carol', 'David']
>>> random.shuffle(people)
>>> people
['Carol', 'Alice', 'David', 'Bob']
>>> random.shuffle(people)
>>> people
['Bob', 'Carol', 'David', 'Alice']
```

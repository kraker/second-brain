# tree

`tree`

| Option | Description                                  |
| ------ | -------------------------------------------- |
| `-a`   | Includes hidden files in the output          |
| `-d`   | Excludes files from the output               |
| `-h`   | Displays file sizes in human-friendly format |
| `-f`   | Prints the full path for each file           |
| `-p`   | Includes file permissions in the output      |

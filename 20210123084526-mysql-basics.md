	### Import a Database ###
	1. Login to mysql client
		 `mysql -u root -p`
	2. Source the database
		 `mysql> source /path/to/file.sql`
	4. Show databases 
		 ```
		 mysql> show databases;
		 +--------------------+
		 | Database |
	   +--------------------+
		 | classicmodels      |
		 | information_schema |
		 | mysql              |
		 | performance_schema |
		 | sys                |
		 +--------------------+
		 ```
		 
	### Useful Commands ###
	* **USE** a specific database:
		`USE db_name;`
	* **SELECT** **FROM** a database:
		`SELECT * FROM table_name;`
